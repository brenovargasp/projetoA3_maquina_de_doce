# Projeto A3 - Máquina de Doce
Repositório destinado para a entrega do Projeto A3 de Teoria da Computação e Compiladores.

# Para dar play no projeto, rode na root os seguintes comandos:

1. cd backend
2. python -m venv .venv
3. .venv\Scripts\Activate.ps1
4. pip install -r requirements.txt
5. cd ..
6. python -m uvicorn backend.app.main:app --reload
